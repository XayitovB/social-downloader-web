# 🌐 Social Downloader Web

Modern web app for downloading videos from TikTok, YouTube, and Instagram.

![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=flat&logo=typescript&logoColor=white)
![React](https://img.shields.io/badge/React-18-61DAFB?style=flat&logo=react&logoColor=black)
![Vite](https://img.shields.io/badge/Vite-646CFF?style=flat&logo=vite&logoColor=white)
![License](https://img.shields.io/github/license/XayitovB/social-downloader-web?style=flat)

## Features
- Download from TikTok, YouTube, Instagram
- Dark / Light mode
- Bilingual UI (English & Indonesian)
- Responsive design (mobile & desktop)
- Serverless backend via Netlify Functions

## Local Development
```bash
npm install
npm run dev
```

## Build & Deploy
```bash
npm run build   # outputs to dist/
# Deploy to Netlify automatically on git push
```

## Stack
`React 18` · `TypeScript` · `Vite` · `Tailwind CSS` · `Framer Motion` · `Netlify`
